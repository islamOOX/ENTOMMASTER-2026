ℹ️ À propos de cette classification

Cette classification agronomique organise les insectes nuisibles selon leurs cultures hôtes principales, permettant une identification rapide sur le terrain. Elle est complémentaire de la classification taxonomique par ordre entomologique disponible sur la page d'accueil.

Les associations espèce–culture sont basées sur les données de l'OEPP Global Database et des publications de l'INRA, avec un focus particulier sur les cultures du bassin méditerranéen.
